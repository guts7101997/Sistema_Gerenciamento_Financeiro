# Sistema_Gerenciamento_Financeiro
Projeto em Pythonde gerenciamento financeiro utilizando POO e bibliotecas. O projeto tem como objetivo com base despesas e receitas inseridas, operações básicas possam ser feitas registrando esses dados, gerando relatórios para datas específicas e podendo criar um arquivos CSV dos dados apresentados.

Foram utilizadas as bibliotecas CSV, para gera o arquivo com extensão de mesmo nome, e Datetime, para registrat a data das transações. O conceito de POO se apresenta na criação das classes como GerenciadorFinanceiro as classes de modelo como Transações ou Receita, sendo aplicados também os conceitos de herança, polimorfismo e clean-code.
